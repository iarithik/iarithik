My expertise is rooted in applying quantum computing to real-world problems, as demonstrated by the following key projects:

*   **Quantum Approaches to Vehicle Routing Problems (VRP):** Formulated QUBO models and implemented both Full Qubo Solver (FQS) and Average Partitioning Solver (APS) using the D-Wave Ocean SDK. I developed a graph-coarsening pipeline to scale VRP instances, significantly improving solver runtime on large graphs.
*   **Quantum Walk and Monte Carlo Simulations:** Explored the difference between classical and quantum simulations of the Galton Board. This project involved processing NASA battery datasets and formulating the problem as a QUBO model to enhance State of Health (SoH) and extend battery life.
